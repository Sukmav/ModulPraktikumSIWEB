# Manajemen Pet Clinic
Website Sistem Manajemen Pet Clinic menggunakan HTML5 dan Bootstrap 5 dengan struktur terdiri dari navbar berwarna hijau dengan kode #175225ca, hero section dengan gradient background dan animasi float pada gambar, statistics section menampilkan 3 kartu data dengan hover effect, produk section berisi 3 item makanan hewan dan mainan hewan, form registrasi dengan 6 field input dan validasi, serta footer berisi jam operasional. CSS menggunakan animasi fadeInUp dan float, button dengan warna hijau custom, form control dengan border hijau. Menggunakan javaScript untuk form submission dengan pesan alert konfirmasi.
